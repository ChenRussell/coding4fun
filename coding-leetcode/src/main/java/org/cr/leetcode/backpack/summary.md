# 背包问题总结
## 1. dp[] 与 dp[][]
    区别在与二维的dp数组存储了整个过程的数据，而一维数组是对上一层对进行覆盖，本质上是一样的。
  
## 2. dp[j]里数据的意义
    每种题目都是判断背包j的大小与给定数组里的一个值(weight)比较,如果能放下，就对其进行赋值，
    至于赋值的含义，不同的题目不一样。
    1. 如果是求背包最大能装多大的重量，dp[j]=max(dp[j],dp[j-wei]+wei)
    2. 如果是求背包里最大能放最大的价值，dp[j]=max(dp[j],dp[j-wei]+value)
    3. 如果是求背包大小为j时，能否装满背包，dp[j]=dp[j]||dp[j-wei]
    4. 如果是求背包大小为j时，有多少种装满的组合，dp[j]=dp[j]+dp[j-wei]
    5. 如果是求背包大小为j时，最少元素个数装满背包，dp[j]=min(dp[j],dp[j-wei]+1)
    ...
    大多数情况下，最优的都是dp[j-wei]+wei这种，但是有的时候最优的是dp[j]，这里要注意不是dp[j-1]!!
    dp[j-wei]表示背包大小为j-wei时所对应的最优解！
## 3. 给定的weight是否能重复使用
    在进行两层for循环时，第一层是对weight数组循环，第二层是对dp数组循环，对dp数组循环时，
    如果是从左到右，表示weight能重复使用；如果是从右到左，表示weight不能重复使用。
    
## 4. 背包问题的几种应用场景
    1. 判断背包能不能装满 -> leetcode416
    2. 计算背包装满所使用的最少元素数量（可重复使用）->leetcode322
    3. 计算背包装满有多少种可能(元素可重复使用，不可重复使用) -> 腾讯笔试第二题，BackPackV,BackPackIV
    4. 计算背包最大放入重量 ->BackPackI
    5. 计算背包最大放入价值 ->BackPackII
